# Herbalytix Shampoo

Premium Herbal Shampoo for Hair Growth and Care.

## Project info

This project is a React application built with Vite, Tailwind CSS, and shadcn/ui.

## How to run locally?

Follow these steps:

```sh
# Step 1: Install the necessary dependencies.
npm install

# Step 2: Start the development server.
npm run dev
```

## Technologies Used

- Vite
- TypeScript
- React
- shadcn/ui
- Tailwind CSS
